# CultureMapNavigator

CultureMapNavigator is an AI-powered workplace communication and collaboration coaching system built with Microsoft Copilot Studio.

The project helps users improve:
- workplace communication
- stakeholder communication
- cross-cultural collaboration
- meeting preparation
- conflict handling
- communication adaptation
- collaboration alignment

---

# CultureCompassTest

CultureCompassTest is the onboarding tool used to generate structured communication baseline profiles.

The tool helps users identify communication tendencies such as:
- communication context preference
- feedback style
- persuasion style
- leadership preference
- disagreement style
- scheduling preference

The generated USER_PROFILE can then be pasted into CultureMapNavigator for personalized communication coaching.

---


# Privacy Philosophy

Communication profiles are intended for temporary conversational context unless an explicit storage workflow is implemented.

The project is designed to support communication coaching and collaboration improvement, not personality diagnosis or HR evaluation.
